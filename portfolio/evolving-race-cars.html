Evolving Race Cars

June 2019

Overview

This project aimed to teach a car to teach itstelf to drive around a race track, and, more importantly, to demonstrate an evolution-like system. Running this program requires the Python modules pygame, numpy, and pickle.


The main.py file loads racetrack1.png and spawns "race cars". Each race car has a randomly generated value for how much it accelerates, decelerates, and turns. It also has randomly generated control rods so that it can determine its output action (steer left, steer right, speed up, or slow down) based on its input values (distance to the black wall from the left, center, and right field of view, as well as its current speed).


Each car in the race is given a score based on the number of green checkpoints it crosses and the number of times it collides with the wall. The best cars "breed" with other ones in order to generate more cars similar to it, and perhaps even better.


Pressing the down arrow key allows the user to download the data of the best car at anytime. The pickle'd file is then read in the test.py file so that the car can be tested on a new track. In the data directory, car1, car2, car3, and car4 are all cars which can be tested. None are perfect, but have been able to successfully complete a new racetrack.


Inspired by Dr. Simon Clark's video.
